# L'IA en 10 minutes

Pas besoin d'être informaticien. Quatre idées suffisent pour bien s'en servir aujourd'hui.

## 1. Ce qu'est un assistant comme Gemini

C'est un programme entraîné sur d'énormes quantités de textes, qui **prédit la
suite la plus probable** de ce qu'on lui écrit. Conséquence pratique : il est
excellent pour **reformuler, structurer, rédiger** — exactement ce dont on a
besoin pour une trace écrite.

## 2. Le prompt, c'est votre consigne

Le **prompt** est le message que vous envoyez. La règle est simple :

!!! tip "Plus la consigne est précise, meilleur est le résultat"
    Donnez le **rôle** (« tu es professeur de physique-chimie »), le **public**
    (« des élèves de 3ᵉ »), le **cadre** (« le programme officiel ») et le
    **format** voulu (« avec définitions et un exemple concret »).

## 3. L'IA peut se tromper avec aplomb

Elle peut inventer une définition, ajouter une notion de lycée, ou affirmer une
bêtise **sur un ton parfaitement sûr**. C'est ce qu'on appelle une *hallucination*.

!!! warning "D'où la règle d'or"
    On ne copie **jamais** une production de l'IA sans la relire. Pour un cours,
    c'est non négociable : **vous validez chaque phrase**.

## 4. Notre arme secrète : cadrer sur le programme

C'est la consigne qui change tout pour nous. En demandant explicitement de
**respecter le programme officiel de 3ᵉ et de n'introduire aucune notion
hors-programme**, on évite le travers le plus courant : un contenu juste, mais
trop avancé pour la classe.

---

!!! note "À retenir avant de commencer"
    **Rôle + public + cadre programme + format → relecture.** C'est toute la
    méthode. On passe à la pratique.
